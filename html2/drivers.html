Drivers

Uitleg

                    De logischelaag en fysiekelaag kunnen opzich niet met elkaar comminuceren. Hiervoor zijn drivers
                    gemaakt. Via drivers kan een operating system signalen sturen naar een hardware component. 
                    Drivers zijn dus erg belangrijk. Voor iedere soort hardware is wel een losse driver nodig. 
                    Voor bijvoorbeeld een beeldscherm is een beeldscherm driver nodig. Een operating system beschikt
                    dus over heel veel verschillende drivers. Als er voor een bepaald soort hardware geen driver beschiktbaar
                    is moet er een update worden uitgevoerd zodat er een driver voor dat soort hardware beschikbaar is.
