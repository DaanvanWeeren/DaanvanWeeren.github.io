Userinterface

Uitleg

                    Een userinterface zorgt er voor dat de gebruiker van een bepaald apparaat daadwerkelijk kan comminuceren met dat apparaat.
                    Een Graphical Userinterface, kort gezegd GUI, is een veel gebruikte vorm van een userinterface. Deze werkt op het zicht 
                    van de gebruiker. Zo is een beeldscherm een voorbeeld van een GUI. Ook moet de gebruiker natuurlijk informatie kunnen doorgeven
                    aan het apparaat. Bij een computer gaat dit bijvoorbeeld via een toetsenbord. De actie de hierbij voorkomen worden vertaald
                    naar events. Deze worden vervolgens weer verwerkt door een Event Handler.
