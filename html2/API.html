API's

Uitleg

                    API (Application Programming Interface) is een voorbeeld van een software-interface. Deze worden gebruikt voor de comminucatie tussen de 
                    logischelaag en de toepassingenlaag. Een API zorgt er bijvoorbeeld voor hoe bepaalde knoppen tijdens het gebruik van
                    bijvoorbeeld een app op het scherm komen. Hierdoor ontstaat er een vorm van consistentie wat natuurlijk erg fijn is voor
                    de gebruiker. Een API kan regelt ook dingen die niet zichtbaar zijn voor de gebruik zoals connectie maken met de wifi.
                    Voor het halen van data uit het internet wordt een web-API gebruikt.
