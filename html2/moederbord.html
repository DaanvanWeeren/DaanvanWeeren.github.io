Het moederbord en de bus

inleiding

                    Het moederbord van bijvoorbeeld een computer is een grote printplaat waarop allemaal hardware componenten van je computer
                    op aangesloten zijn. Op deze printplaat zitten allemaal bussen. Dit is een voorbeeld van een hardware-interface. 
                    Eigenlijk is het moederbord dus een grote interface tussen bijna alle hardware componenten van je computer.

Het moederbord

                    Zoals in de inleiding al is gezegd is het moederbord eigenlijk vooral een grote interface tussen allemaal hardware componenten.
                    Zo kunnen de processor en RAM dus met elkaar comminuceren. 
                    Een moederbord is dus erg belangrijk niet alleen om deze reden, maar ook omdat zo informatie van bijvoorbeeld sensoren naar actuatoren
                    vervoerd kan worden. Belangrijk is dus dat de hardware componenten over het algemeen niet met elkaar verbonden zijn,
                    maar via het moederbord verbonden zijn.

de bus

                    Een bus kan je als een soort lijntje op een printplaat zien zitten. Zo een bus zorgt ervoor dat er impulsen van het ene hardware component 
                    naar het andere hardware component vervoerd kan worden. Er zitten ook bussen tussen de processor en RAM. Deze bussen zijn te verdelen in 
                    drie soorten: de data bus, de control bus en de adress bus. Via een adress bus wordt informatie gestuurd naar RAM welke informatie nodig is.
                    Op het moment dat de control bus een signaal door geeft kan de data via de data bus verstuurd worden. Let op dat de processor zowel informatie 
                    kan verkrijgen uit RAM en informatie kan opslaan in RAM. In een data bus kan de stroom dus twee kanten oplopen. Bij een adress bus
                    is dit niet het geval, omdat dit alleen een plaats in RAM aan geeft dus loopt de stroom over adress bus alleen naar RAM toe.
