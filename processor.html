De processor

Inleiding

                    De kern van je computer bestaat uit de processor en de memery unit. Een processor kan heel snel taken
                    uitvoeren, maar kan niet multitasken. Ook bestaat de processor zelf uit nog meerdere onderdelen. Hoe dat allemaal werkt kan je in dit
                    hoofdstuk lezen.

De ALU en de control unit

                    De processor moet altijd zo snel mogelijk een berekening doen met een verzameling aan data. Hiervoor zijn twee verschillende onderdelen nodig.
                    Dat is de arthimatic logic unit (ALU) en de control unit. Het doel van de control unit is om te bepalen wat er met de data verkregen uit RAM 
                    moet gebeuren. Vervolgens kan de ALU de instructie van de control unit uitvoeren en de nieuwe data terug geven aan de control unit.
                    Nu kan de control unit dit bijvoorbeeld terug zetten in RAM.

Verwerkingssnelheid en kloksnelheid

                    Bij de processor hoort een FDE-cycle. dit staat voor fetch (het verkrijgen van data), decode (wat moet er met de data gedaan worden)
                    en excute (het uitvoeren van de instructies). Hoe snel zo een cycle wordt uitgevoerd wordt de verwerkingssnelheid genoemd.
                    Deze wordt gemeten in mips (millions of instructions per second).
                    Er is ook nog een andere snelheid. Hier voor is het handig om te weten wat de verschillende bussen doen.
                    Iedere keer dat de control bus een signaal stuurt kan er data naar de processor toe lopen. de hoeveelheid impulsen van de control bus
                    wordt de kloksnelheid genoemd. Deze is gemeten in herts (Hz). De kloksnelheid is hoger dan de verwerkingssnelheid, omdat iedere FDE-cycle 
                    uit een aantal impulsen van klokchip bestaat.
