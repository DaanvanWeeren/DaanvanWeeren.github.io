Het geheugen

inleiding

                Het geheugen van een computer bestaat uit verschillende componenten. Zo heb je het permanente geheugen, het RAM en het ROM
                Elke vorm van geheugen heeft zijn voor- en nadelen. Die gaan we in dit paragraaf bespreken.

Het permanente geheugen

                Het voordeel aan het permanente geheugen is dat er veel in kan worden opgeslagen. Daarnaast zullen de gegevens in
                het permanente geheugen ook niet zo maar verwijderd worden als de computer wordt uitgeschakelt. Dit is natuurlijk erg handig.
                Een groot nadeel is wel dat er erg veel tijd nodig is om data uit het permanente geheugen te laden. Zo zie je ook dat de 
                eerste computers erg langzaam waren doordat ze alleen een permament geheugen hadden en dus duurde het lang voordat er een programma
                was geladen op deze computer.

RAM

                In 1945 kwam Jogn von Neumann met een idee om een computer sneller te laten werken. Dit wordt de Von Neumann architectuur genoemd.
                Bij de Von Neumann architectuur kreeg je de samenwerking tussen de processor en RAM (Random Access Memory) geheugen. 
                Doordat RAM dichterbij de processor zit werkt het veel sneller waardoor programma's sneller geladen konden worden. Wel heeft RAM 
                zijn nadelen. Zo heeft het bijvoorbeeld minder ruimte dan het permamente geheugen en als de spanning weg valt gaat de data die in RAM
                staat opgeslagen verloren. Meestal staan er in RAM dingen als instructies, getallen, letters en adressen opgeslagen.

ROM

                In ROM staat informatie opgeslagen voor de Basic Input/Output System (BIOS). Hierin staan een paar instelling 
                waarmee een paar testen worden uitgevoerd voordat de computer de rest van de data laad. Het is dus belangrijk dat
                de data op ROM niet verloren gaat als de spanning weg valt. Ook is ROM goed beveiligd waardoor niet alle software zo maar toegang 
                heeft tot deze informatie.
