de logischelaag

Inleiding

                    De logischelaag is erg belangrijk voor een device. Meestal bestaat de logischelaag uit een operating system.
                    Voorbeelden van een operating system zijn IOS of android. Een operating system heeft drie taken. 
                    Het beheer van randapparatuur, bestandsbeheer en taakbeheer.

Het beheer van randapparatuur

                    Een operating system zorgt ervoor dat het maken van een app veel makkelijker wordt. Als er namelijk geen 
                    operating system zou zijn dan zou een app voor ieder soort randapparatuur worden gemaakt. Dat zou er onhandig zijn,
                    want dan zou iedere app designer ook een hardware specialist moeten zijn. Door de komst van een operating system is
                    dit niet meer nodig, want nu stuurt het operating system de hardware aan door middel van drivers.
                    De app hoeft nu alleen aangespast te zijn aan het operating system. Zo werkt een app gemaakt voor IOS niet op een android
                    operating system.

bestandsbeheer

                    Een operating system is ook verantwoordelijk voor de opslag van bestanden. Als je iets wilt verwijderen moet het
                    operating system wel weten waar het is opgeslagen anders verdwijrt het misschien verkeerde bestanden en dat wil je niet hebben.
                    Ook als je iets opslaat zorgt het operating system ervoor dat het zo dicht bij elkaar mogelijk wordt opgeslagen. 
                    Stel je wilt een programma opslaan dat 100 kilobyte is maar er zijn geen plekken vrij waar 100 kilobyte kan worden opgeslagen.
                    Dan zal het operating system ervoor zorgen dat deze 100 kilobyte aan bestand zo dicht bij elkaar mogelijk wordt opgeslagen.

taakbeheer

                    Als laatste bepaald het operating system ook hoeveel cpu tijd elke taak krijgt. Hierdoor blijft alles goed werken
                    en gaan er geen dingen door elkaar heen lopen. Voor informatie over de cpu kan je het hoofdstuk over de processor lezen.
